# 자기소개 페이지 공유 방법 가이드

## 방법 1: GitHub Pages (무료, 추천) ⭐

### 장점
- 완전 무료
- 전 세계 어디서나 접근 가능
- URL이 깔끔함 (예: username.github.io/프로젝트명)
- 자동 업데이트 가능

### 단계별 방법

1. **GitHub 계정 만들기** (없는 경우)
   - https://github.com 접속
   - Sign up 클릭하여 계정 생성

2. **새 저장소(Repository) 만들기**
   - GitHub 로그인 후 우측 상단 `+` 버튼 → `New repository`
   - Repository name: `my-portfolio` (원하는 이름)
   - Public 선택 (Private는 유료)
   - `Create repository` 클릭

3. **파일 업로드**
   - 생성된 페이지에서 `uploading an existing file` 클릭
   - `자기소개_손상기.html`과 `P.png` 파일을 드래그 앤 드롭
   - `Commit changes` 클릭

4. **GitHub Pages 활성화**
   - 저장소 페이지에서 `Settings` 탭 클릭
   - 왼쪽 메뉴에서 `Pages` 클릭
   - Source에서 `Deploy from a branch` 선택
   - Branch를 `main` 또는 `master` 선택
   - `Save` 클릭

5. **공유 URL 확인**
   - 몇 분 후 `Settings` → `Pages`에서 URL 확인
   - 예: `https://사용자명.github.io/my-portfolio/자기소개_손상기.html`
   - 이 URL을 다른 사람에게 공유하면 됩니다!

---

## 방법 2: Netlify Drop (가장 쉬움) ⭐⭐⭐

### 장점
- 계정 없이도 사용 가능
- 드래그 앤 드롭만으로 완료
- 자동으로 URL 생성

### 단계별 방법

1. **Netlify Drop 접속**
   - https://app.netlify.com/drop 접속

2. **파일 업로드**
   - `자기소개_손상기.html`과 `P.png` 파일을 폴더에 함께 넣기
   - 폴더 전체를 Netlify Drop 페이지에 드래그 앤 드롭

3. **URL 확인 및 공유**
   - 자동으로 생성된 URL 확인 (예: `random-name-123.netlify.app`)
   - 이 URL을 다른 사람에게 공유

**참고**: Netlify Drop은 임시 URL이므로, 영구적으로 사용하려면 Netlify 계정을 만들어야 합니다.

---

## 방법 3: 파일 직접 공유

### 방법 3-1: 이메일 첨부
- `자기소개_손상기.html`과 `P.png` 파일을 함께 압축
- 압축 파일을 이메일로 전송
- 받는 사람은 압축 해제 후 HTML 파일을 브라우저로 열기

### 방법 3-2: 클라우드 스토리지
- **Google Drive / OneDrive / Dropbox** 등에 파일 업로드
- 공유 링크 생성하여 전송
- 받는 사람은 파일 다운로드 후 열기

**주의사항**: 이미지 파일(`P.png`)도 반드시 함께 공유해야 합니다!

---

## 방법 4: 로컬 서버 실행 (개발/테스트용)

### Python 사용 (가장 간단)
1. 파일이 있는 폴더로 이동
2. 명령 프롬프트에서 실행:
   ```bash
   python -m http.server 8000
   ```
3. 브라우저에서 `http://localhost:8000/자기소개_손상기.html` 접속
4. 같은 네트워크에 있는 사람은 `http://본인IP주소:8000/자기소개_손상기.html`로 접속 가능

---

## 추천 순서

1. **빠르게 공유하고 싶다면**: Netlify Drop (방법 2)
2. **영구적으로 공유하고 싶다면**: GitHub Pages (방법 1)
3. **소수의 사람에게만 공유**: 파일 직접 공유 (방법 3)

---

## 주의사항

1. **이미지 파일 필수**: `P.png` 파일도 반드시 함께 공유해야 합니다.
2. **파일 경로**: HTML 파일과 이미지 파일이 같은 폴더에 있어야 합니다.
3. **파일명**: 공유 시 파일명이 한글이어도 대부분 문제없지만, 영문으로 변경하는 것을 권장합니다.

---

## 파일명 영문 변경 (선택사항)

더 나은 호환성을 위해 파일명을 영문으로 변경할 수 있습니다:
- `자기소개_손상기.html` → `index.html` 또는 `portfolio.html`
- `P.png` → `profile.png`

HTML 파일 내부의 이미지 경로도 함께 수정해야 합니다:
```html
<img src="profile.png" alt="손상기 프로필 사진" class="hero-image">
```
